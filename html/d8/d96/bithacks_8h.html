◆ bsf()

ssize_t bsf ( const void * data,
size_t start_bit,
size_t len,
bool reverse )

Find the first bit at 1 or 0 in a sequence of bits.

Parameters
[in]datathe memory to scan
[in]start_bitoffset of the first bit after data
[in]lenmaximum number of bits to scan after start_bit
[in]reverseif reverse, look for the first 0 instead of the first 1
Returns
-1 if no 1 (or 0 if reverse was set) was found, else the offset of the first bit of the given value relative to start_bit.
