int buffer_increment ( char * buf,
int len )

Increment last counter in a buffer.

buf points to the start of the buffer. len can be negative, in this case len=strlen(buf)

Examples : "000" => "001" "999" => "000" "foobar-0-01" => "foobar-0-02" "foobar-0-99" => "foobar-0-00"

Returns
1 if an overflow occurred, 0 otherwise
