static int unpack_struct ( iop_json_lex_t * ll,
const iop_struct_t * desc,
void * value,
bool prefixed )
static

Unpack a json struct into a C struct.

Parameters
[in]llthe lexer.
[in]desca descriptor of the output structure. Must be NULL if value is NULL, which will skip all the fields (as we don't know the output structure layout).
[out]valuea pointer to the structure to fill. Can be NULL iff desc is NULL.
[in]prefixedif set to true, the function assumes that it has to parse a value with the prefix syntax: @member value { ... }
