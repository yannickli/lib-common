◆ hpack_decoder_write_hdr()

int hpack_decoder_write_hdr ( hpack_dec_dtbl_t * dtbl,
hpack_xhdr_t * xhdr,
byte * out,
int * keylen )

Decode and write the header line referenced by xhdr into out.

Returns
the number of bytes written, or -1 on error.
Note
the hdr line written has the format 'KEY(-OF-KEYLEN): VALUE\r
'. On success, keylen is the length of key part, i.e, the offset of ':' in the written hdr line. So, it is up to the caller to ensure that out has at least 4 bytes more than the value return by hpack_decoder_extract_hdr.
