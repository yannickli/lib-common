const popt_t *nonnull opts,
int len )

Append options into a vector of popt_t.

Make sure that the vector ends with a OPT_END().

This function will ignore provided OPT_END() (when given as last element) as it handles the terminating option by itself. It will also remove the current terminating OPT_END() (if any) before appending new options.
