◆ fd_get_path()

ssize_t fd_get_path ( int fd,
char buf[],
size_t buf_len )

Get the path of the file opened by that file descriptor.

In case of success this function guarantees the buffer is terminated by a nul byte. It only works if the fd points to a regular file or a directory that hasn't been moved or renamed since it was opened.

Parameters
fdThe filedescriptor to inspect.
bufA buffer that will receive the path.
buf_lenThe size of buf.
Returns
A negative value if the filename cannot be retrieved or if the buffer is too short. Or the size of the string stored in the buffer in case of success (not counting the trailing nul byte).
Warning
this function is expensive. It performs up to 3 system calls.
