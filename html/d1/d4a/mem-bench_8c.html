mem_bench_t * mem_bench_init ( mem_bench_t * sp,
lstr_t type,
uint32_t period )

Initialize mem_bench object.

The mem_bench object is initialized to dump into a file each period iterations. The filename is "./mem.[\p name].[pid].[address of \p sp]". "[type].[address of sp] is also used for logger name.

Parameters
[in]typeName of mem_bench object.
[in]periodLogging period. 0 means no logging.
