static void qvector_optimize ( qvector_t *nonnull vec,
size_t v_size,
size_t v_align,
size_t ratio,
size_t extra_ratio )
inlinestatic

optimize vector for space.

Parameters
[in]vecthe vector to optimize
[in]v_sizesizeof(val_t) for this qvector
[in]ratiothe ratio of garbage allowed.
[in]extra_ratiothe extra size to add when resizing.

If there is more than vec->len * (ratio / 100) empty cells, the array is resize to vec->len + vec->len * (extra_ratio / 100).

In particular, qvector_optimize(vec, ..., 0, 0) forces the vector allocation to have no waste.
