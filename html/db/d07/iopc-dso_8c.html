Build an IOP DSO.

Parameters
[in]pfxdirprefix directory of the IOP file to compile
[in]display_pfxset to false if only the relative part of the files should be printed in case of error
[in]iopfilethe IOP file to compile; this path must be relative to pfxdir
[in]enva map of buffered IOP files (dependencies); the paths of these dependencies must be relative to pfxdir
[in]outdirthe absolute path of the directory to store the IOP DSO file (outdir/pkgname.so)
[out]errbuffer filled in case of error
Returns
0 if ok, -1 if the build failed
