◆ debug_stack_cb_f

typedef void debug_stack_cb_f(int fd, data_t data)

Function executed when writing the .debug file.

This callback is called when a crash happens in the debug stack scope where the callback has been registered.

Warning
It should write the user data using dprintf. Please notice that our custom formats (%*pM, pL, %*pS, ...) wont work in this context.
This code is called in the context of a crash, some contextual data may be invalid or corrupted.
Do not use malloc or functions that could use malloc in the callback. It could result in deadlocks.
