 run (self, test)
 

Detailed Description

 split test suite output by group

By default unittest as the following test tree:
  suite
    suite1
      test1
      test2
    suite2
      test3
But z format is defined as follow:
  suite
    groupe1
      test1
      test2
    groupe2
      test1
