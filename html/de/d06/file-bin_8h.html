Iterates on each record of a file.

Create a loop to iterate on each record (starting from the current reading position) of the file. This macro should be used when it is possible instead of having a manual loop built with t_file_bin_get_next_record, to avoid the loading of the entire file in memory.

Note that the memory pointed by 'entry' lstr_t becomes invalid on the next iteration.

Parameters
[in]fileThe binary file to read.
[in]entryThe name of the lstr_t that will be created and which will contains each record.

__must_check__ int _file_bin_seek ( file_bin_t * file,
off_t pos )

Move the current file position to the one given as argument.

Be careful as this function is kind of tricky. It will move the current reading position of a file_bin (which MUST have been opened with file_bin_open before) to the one given as argument. If the new position is not the beginning of a slot or the beginning of an entry, it will break the record reading for the slot you are in.

Parameters
[in]filethe file_bin concerned by the seek.
[in]posthe new reading position.
Returns
0 on success, a negative value otherwise.
