◆ __dlist_repair()

static void __dlist_repair ( dlist_t * e)
inlinestatic

Repair a dlist after its head or one of its elements address changed.

Can happen in case of copy or reallocation: the 'prev' address of the 'next' element and the 'next' address of the 'prev' element are broken because they still have the old pointer value. This function refreshes both.

XXX Won't work on empty lists or detached elements: for a detached dlist, 'next' and 'prev' contain the old address of the dlist_t and cannot be dereferenced.

Example:

bool was_empty = dlist_is_empty(&ptr->link); ptr = realloc(ptr, 1209); if (was_empty) { dlist_init(&ptr->link); } else { __dlist_repair(&ptr->link); }

Parameters
[in,out]eElement or list head that got its address changed.
