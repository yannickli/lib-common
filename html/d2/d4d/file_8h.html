int write_in_tmp_file ( char * file_path,
const char * data,
int len,
sb_t * err )

Create a temporary file and write data in it.

Standard 'unlink' method should be called on the path given in argument in case of success (once you no longer need the file).

Parameters
[in]file_pathpath of the file. It will create with the method mkstemp. As mentioned, in the doc of this method, the last six characters of template must be "XXXXXX" and these are replaced with a string that makes the filename unique.
[in]datathe data to write.
[in]lenlength of data.
[out]errerror on creating file or writing data.
Returns
0 on success, -1 the data has not be written.
