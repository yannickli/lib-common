void bb_init_sb ( bb_t * bb,
sb_t * sb )

Initialize a bit-buffer from a string-buffer.

The bit-buffer takes ownership of the memory and resets the sb so we're guaranteed no dangling pointers will remain in the sb if the bit-buffer performs a reallocation sometime in the future.

◆ bb_transfer_to_sb()

void bb_transfer_to_sb ( bb_t * bb,
sb_t * sb )

Transfer ownership of the memory to a sb.

The bit-buffer is reset during the operation and the sb gains full ownership of the memory.

This operation does not preserve the bit-buffer alignment since sb don't support custom alignment.
