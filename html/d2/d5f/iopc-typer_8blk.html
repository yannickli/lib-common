◆ iopc_pkg_get_deps()

void iopc_pkg_get_deps ( iopc_pkg_t * pkg,
unsigned flags,
qv_t(iopc_pkg) * t_deps,
qv_t(iopc_pkg) * t_weak_deps,
qv_t(iopc_pkg) * i_deps )

For a given IOPC package, list the other IOPC packages it depends on.

Parameters
[in]flagsFlags from iopc_pkg_get_deps_flags_t.
[out]t_depsDependencies for which complete types definitions will be needed. These complete definitions are meant to be found in header files "<package>-t.iop.h".
[out]t_weak_depsDependencies for which only forward types declarations are needed (for example when a struct attribute is a pointer). These definitions are meant to be found in header files "<package>-t.iop.h".
[out]i_depsDependencies induced by interfaces definitions. Interfaces and modules stuff are not dumped in the same header as structs/union/classes so their dependencies are listed separately.
