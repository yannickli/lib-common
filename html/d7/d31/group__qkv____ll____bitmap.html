Detailed Description

QPS Bitmap

This bitmap implementation is a 3-level trie mapping a key to a bit. It supports both simple bitmaps and "nullable" bitmap. The nullable implementation associate a pair of bit to each key with the following possible combinations:

  • 00: NULL
  • 01: unused combination
  • 10: bit set at 0
  • 11: bit set at 1
