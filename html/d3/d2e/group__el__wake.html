#include <el.h>

Register a waker.

A waker is an event loop object that get manually triggered. One can call el_wake_fire() from another thread in order to wake up an event loop waiting on the waker.

This is a low level primitive provided in order to build higher-level infrastructure. Look at thr_queue and thr_queue_main_g before using a waker by hand.
