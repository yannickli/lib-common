typedef ssize_t formatter_f(int modifier, const void *val, size_t val_len, FILE *stream, char *buf, size_t buf_len)

Formatter function type.

A formatter is called when, for a formatter f, the format %*pf is encountered in the format string. The formatter is then called with the parameters passed as argument of that particular format as well as the modifier.

A formatter can either write in a stream or in a buffer, in both cases, it must return the size of the formatted object if it had enough place to write in it. The function must check if the output is supposed to be a stream or a buffer based on the presence of a stream.

The buffer may not be large enough to write the received data, and the formatter is not responsible for the appending of a terminating zero.

Parameters
[in]modifierThe modifier that triggered the call
[in]valThe pointer received as parameter.
[in]val_lenThe length received as parameter.
[out]streamIf the formatter is expected to write on a stream, this is the destination stream.
[out]bufIf the formatter is expected to write in a buffer, this is the destination buffer.
[out]buf_lenThe size of the buffer.
Returns
The size required to format the val (not including the trailing zero) or -1 in case of error.

Pointer Formatter function type.

A pointer formatter is called when, for a formatter f, the format pf is encountered in the format string. The formatter is then called with the parameters passed as argument of that particular format as well as the modifier.

see formatter_f description for more details.

Parameters

[in]valThe pointer received as parameter.
[out]streamIf the formatter is expected to write on a stream, this is the destination stream.
[out]bufIf the formatter is expected to write in a buffer, this is the destination buffer.
[out]buf_lenThe size of the buffer.
Returns
The size required to format the val (not including the trailing zero) or -1 in case of error.
